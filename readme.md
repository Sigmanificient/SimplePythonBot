Make sure to follow the requirements in `requirements.txt`

The bot token should be in a `.env` file such as the `.env.template` file:
```dotenv
TOKEN=YOUR_BOT_TOKEN
```

## Default commands
-   `help`, `bot`, `invite`, `ping`, `load`, `unload`, `reload`
